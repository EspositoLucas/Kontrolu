Guía de Botones del Simulador

Configurar Simulación

Este botón permite ajustar los parámetros básicos de la simulación:

  • Tiempo total: Duración completa de la simulación
  • Variable a controlar: El valor objetivo que deseas mantener
  • Muestreo de datos: Tiempo entre cada paso de la simulación
  • Duración de ciclos: Tiempo real que toma cada ciclo
  • Simulación Precisa: Activa un modo más preciso pero que requiere más recursos

Archivo

Desde este menú puede gestionar sus archivos de trabajo:

  • Nuevo Archivo: Comienza un nuevo proyecto desde cero
  • Abrir Archivo: Carga un proyecto existente
  • Guardar Archivo: Guarda tu trabajo actual

Editar JSON

Esta herramienta permite:

  • Ver y modificar los datos del modelo en formato JSON
  • Copiar el contenido al portapapeles
  • Cargar datos desde archivos JSON externos
  • Guardar tus modificaciones

Copiar y Guardar Diagrama

Con un solo clic, puede:

  • Capturar el diagrama actual de tu sistema
  • Guardarlo como imagen PNG en tu dispositivo
